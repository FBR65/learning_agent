❓ Hilfe & Anleitung

🚀 Erste Schritte

  1. 📁 Upload: Lade dein Lernmaterial hoch (CSV-Format ist am einfachsten)
  2. 📚 Szenarien: Wähle dein Lernset aus
  3. 🎓 Lernen: Starte eine Lernsession
  4. 🔄 Wiederholung: Wiederhole fällige Karten

📝 Dateiformat für Upload

CSV-Format (empfohlen):

Frage,Antwort,Kategorie,Schwierigkeit
"Was ist die Hauptstadt von Deutschland?","Berlin","Geografie",1
"Wie lautet die Formel für Wasser?","H2O","Chemie",1

💡 Tipps:

  • Erste Zeile muss die Spaltenüberschriften enthalten
  • Schwierigkeit: 1 = leicht, 2 = mittel, 3 = schwer
  • Kategorien helfen beim Organisieren der Fragen
  • Anführungszeichen verwenden, wenn Text Kommas enthält

🔄 Spaced Repetition System

So funktioniert das Lernsystem:

  • Neue Karten: Werden nach 1 Tag wiederholt
  • Richtige Antworten: Längere Intervalle (1 → 6 → 13+ Tage)
  • Falsche Antworten: Sofortige Wiederholung
  • Bewertung: 0-2 = schwer, 3-5 = gut bis perfekt

Das System passt sich automatisch an deine Leistung an!

🎯 Lerntipps

  • Regelmäßig lernen: Lieber 10 Minuten täglich als 1 Stunde einmal pro Woche
  • Ehrlich bewerten: Bewerte deine Antworten realistisch
  • Bei Unsicherheit: Wähle eine niedrigere Bewertung
  • Pausen machen: Bei Müdigkeit ist das Lernen weniger effektiv
